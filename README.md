# Winter-Antics for Minecraft 1.21.11, NeoForge 21.11.0-beta

- Crawling across snow has a chance to leave a Snow Angel behind
- New Blizzard block allows for generating snow anywhere in the world
- New Ornament item can be placed on the ground as a trap or hung on trees as a decoration
- Ornaments can also be thrown as a projectile
- Creating a snow golem with Blizzard blocks creates a Blizzard golem which attacks quickly
